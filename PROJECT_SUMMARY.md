## 🎨 Customization Guide

### Update Personal Info
Edit these components:
- `Hero.tsx` - Name, titles, social links
- `About.tsx` - Summary, education
- `Skills.tsx` - Add/remove skills
- `Projects.tsx` - Update projects
- `Contact.tsx` - Contact details

### Change Colors
Edit `tailwind.config.ts`:
```typescript
neon: {
  blue: '#00f0ff',    // Primary
  purple: '#bf00ff',  // Secondary
  pink: '#ff006e',    // Accent
  green: '#00ff88',   // Success
}
```

### Modify Animations
Edit component files - all use Framer Motion:
```typescript
<motion.div
  initial={{ opacity: 0, y: 50 }}
  animate={{ opacity: 1, y: 0 }}
  transition={{ duration: 0.8 }}
>
```

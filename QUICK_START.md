## 🏃 Run It Now

### 1. Development Server

```bash
cd /workspaces/default/code
pnpm dev
```

The portfolio will open at `http://localhost:5173`

### 2. Build for Production

```bash
pnpm build
```

Output goes to `dist/` folder

### 3. Preview Production Build

```bash
pnpm preview
```

## 🌐 Deploy in 5 Minutes

### Easiest: Vercel

```bash
# Install Vercel CLI
pnpm add -g vercel

# Deploy
vercel
```

Follow prompts - your site goes live instantly!

### Also Easy: Netlify

1. Run `pnpm build`
2. Go to [netlify.com](https://netlify.com)
3. Drag and drop the `dist` folder
4. Done! ✨

### Change Colors

Edit `tailwind.config.ts`:

```typescript

  blue: '#00f0ff',    // Change this
```

## 🆘 Need Help?

### Common Commands

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production  
pnpm build

# Preview build
pnpm preview

# Type check
pnpm exec tsc --noEmit
```

### Issues?

1. Clear cache: `rm -rf node_modules pnpm-lock.yaml && pnpm install`
2. Check browser console for errors
3. Verify Node.js version: `node -v` (should be 18+)

## 🚀 Deploy in 5 Minutes

### Option 1: Vercel (Easiest)
```bash
pnpm add -g vercel
vercel
```

### Option 2: Netlify
```bash
pnpm build
# Then drag and drop the 'dist' folder to netlify.com
```

### Option 3: GitHub Pages
```bash
pnpm add -D gh-pages
# Update vite.config.ts base path
pnpm deploy
```

Full deployment instructions: See `DEPLOYMENT.md`
